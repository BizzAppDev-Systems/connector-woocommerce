Key Features of the Woocommerce Connector
  • Import Partners.
  • Import Products, Product Categories, and Product Attributes.
  • Import Sale orders.
  • Exporting Sale order status and Tracking Information to Woocommerce.

Product Management
To import Attributes and their Values, click the "IMPORT PRODUCT ATTRIBUTES" button. For importing Categories, use the
"IMPORT PRODUCT CATEGORIES" button, and to import Products from Woocommerce, simply click the "IMPORT PRODUCTS"
button.

Sale Order Management
Orders present in Woocommerce.

Click the "IMPORT SALE ORDERS" button to import orders.

Imported Sale Order view.

In Woocommerce backend, when you enable "Mark Order Completed On Delivery" the "Send Tracking Information" will be
visible.If both the boolean are True, the Sale Order will be set to the 'Completed' status in Woocommerce, and it will also
send the 'Tracking Information' to the Woocommerce.

On Delivery Order, Tracking Reference value is exported on Woocommerce as a shipment tracking.

After validating the delivery order, the "EXPORT DELIVERY STATUS" button will be visible in the sale order. Clicking this
button will mark the order as completed in Woocommerce.
