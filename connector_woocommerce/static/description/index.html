Module Connector Woocommerce allows you to integrate Woocommerce with odoo. This module provides various functionalities to efficiently manage Woocommerce data and syncronize it.

Highlighted features of connector Woocommerce mentioned below:
  1. Import Partners.
  2. Import Products, Product Categories and Product Atrributes.
  3. Import Sale orders.
  4. Advanced configurations for importing Partners, Products, and Sale orders with criteria.
  5. Exporting Sale order status and tracking reference to Woocommerce.

Backend Views:
Woocommerce Backend with required credentials in Test mode.

Woocommerce Backend in production mode.

Woocommerce backend Import functionality tab.

Woocommerce backend Advanced Configuration tab.

Partner Management:
Customers Present in Woocommerce.

If the 'Allow Partners without Email' boolean option is True, it permits the import of partners even if they do not have an
associated email address.

Click on the 'Import Partners' button to import Partners.

Imported Partner view.

Product Management:
Product present in Woocommerce.

Enabling the "Allow Partners without SKU" boolean setting permits the import of products lacking SKUs. Additionally, when
the "Product Category" option is chosen at the backend level, products will be imported into the specified category.

To import Attributes and their Values, click the 'Import Product Attributes' button. For Categories, use the 'Import Product Categories' button, and to import Products from Woocommerce, simply click the 'Import Products' button.

Imported Product view.

Product categories present in Woocommerce.

Access Woocommerce categories by clicking the 'Woocommerce Product Categories' in the backend menu.

Imported Woocommerce Product Category view.

Product Attributes with Values present in Woocommerce.

Imported Product Attribute list view.

Imported Product Attribute form view and additionally, you can retrieve updated information on attributes and their values
by clicking 'Import Attribute Value'.

Sale Order Management:
Orders present in Woocommerce.

Click the 'Import Sale Orders' button to import orders.

Imported Sale Order form view.

When you enable 'Mark Order Completed On Delivery' the 'Send Tracking Information' will be visible.If both the boolean are
True, the Sale Order will be sent to the 'Completed' status in Woocommerce, and it will also send the 'Tracking Reference' at
the Delivery Order level to Woocommerce.

Input tracking reference in delivery order to export it as shipment tracking for Woocommerce.

After validating delivery order to "Done" state,"Export Delivery Status" button will be visible in sale order and by clicking it
we can make order completed in Woocommerce.
