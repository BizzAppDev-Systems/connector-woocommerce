The Module Connector Woocommerce allows you to integrate Woocommerce with Odoo, providing various functionalities for efficient Woocommerce data management and synchronization.

Key Features of the Woocommerce Connector:
  1. Import Partners.
  2. Import Products, Product Categories and Product Attributes.
  3. Import Sale orders.
  4. Exporting Sale order status and Tracking Information to Woocommerce.

Partner Management:
Customers Present in Woocommerce.

Product Management:
To import Attributes and their Values, click the "IMPORT PRODUCT ATTRIBUTES" button. For importing Categories, use the
"IMPORT PRODUCT CATEGORIES" button, and to import Products from Woocommerce, simply click the "IMPORT PRODUCTS" button.

Product categories present in Woocommerce.

Access Woocommerce categories by clicking the "Woocommerce Product Categories" in the backend menu.

Imported Woocommerce Product Category view.

Imported Product Attribute list view.

Imported Product Attribute form view and additionally, you can retrieve updated information of attribute values by clicking

Sale Order Management:
Orders present in Woocommerce.

Click the "IMPORT SALE ORDERS" button to import orders.

Imported Sale Order view.

In Woocommerce backend, when you enable "Mark Order Completed On Delivery" the "Send Tracking Information" will be
visible.If both the boolean are True, the Sale Order will be set to the 'Completed' status in Woocommerce, and it will also
send the 'Tracking Information' to the Woocommerce.

On Delivery Order, Tracking Reference value is exported on Woocommerce as a shipment tracking.

After validating the delivery order, the "EXPORT DELIVERY STATUS" button will be visible in the sale order. Clicking this button
will mark the order as completed in Woocommerce.
